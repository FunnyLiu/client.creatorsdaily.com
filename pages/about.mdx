# 关于创造者日报

> 唯有创造才是快乐。只有创造的生灵才是生灵。
> 
> *`Only create is happy。 Only by creating the creatures are creatures。`*
>
> `-- Romain Rolland`

创造者日报是一个创造者社区，这里是爱意满满的作品展示平台，希望打造一个成功的 ProductHunt 中文版社区。

### 角色

在这里，用户有两个角色，`“发现者”`和`“创造者”`。

将产品推荐到创造者日报的，即为`“发现者”`，发现者不必一定是产品的拥有者或创造者，可以仅仅是分享发现到的好产品。

而`“创造者”`便是产品的创造者和拥有者，拥有产品的最高权限。

两个角色都会有特殊的身份标签。
